
# Realtime Collaborative Code Editor

A realtime code editor where we can create rooms, join rooms using room Id,leave rooms, realtime code sync among all the clients and also shows all the connected users.All the users in the room will be notified as soon as any user joins the room or leaves the room in realtime.


## Tech Stack

**Client-side:** HTML, CSS, Javascript, ReactJS

**Server-side:** Node, Express, Socket.io


## Features

- Realtime code sync among all the members in the room
- Code editor with auto completion features
- Unique room ID generation
- Notifications when anyone in the room enters or leaves
- Shows the list of all the users present in the room

## Run Locally

Go to the project directory

```bash
  cd ecommerce-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server.js
```
